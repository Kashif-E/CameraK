# Code of Conduct

CameraK is a community project. We expect all participants to be respectful and constructive.

## Our Standards

We value:

- Respectful, inclusive, and professional behavior
- Clear, constructive feedback
- Collaboration and empathy

Unacceptable behavior includes:

- Harassment, discrimination, or personal attacks
- Trolling, insulting remarks, or disruptive behavior
- Publishing private information without consent

## Scope

This Code of Conduct applies to all project spaces, including GitHub issues, pull requests, discussions, and any official community channels.

## Enforcement

Maintainers may remove, edit, or reject contributions that violate this Code of Conduct. Repeated or severe violations may result in a temporary or permanent ban from the project.
